h2. About

p. "Textile":https://textile-lang.com/ is a markup language (like Markdown) for formatting text in a blog or a content management system (CMS).
"Textile-js":https://github.com/borgar/textile-js is a JavaScript port of Textile created by "Borgar":https://github.com/borgar.

p. This is a sample blog web application of  "Next.js":https://nextjs.org/ that implements and extends 
"Textile-js":https://github.com/borgar/textile-js with post-process hooks as extensions or visitors using "Hypertext Abstract Syntax Tree (hast)":https://github.com/syntax-tree/hast.
